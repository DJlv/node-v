组件使用说明 ​

本站 Markdown 支持 Obsidian 双链嵌入、Callout 提示块 与少量 Vue 组件标签。资源文件统一放在 docs/ 下，与当前文档同级目录组织。

资源目录约定 ​

与某篇文档 docs/章节/文章.md 同级的资源目录：

docs/章节/
├── 文章.md
├── img/          ← 图片（png、jpg、gif、webp、svg 等）
├── excel/        ← 表格（xlsx、xlsm、xls）
└── files/        ← 附件（pdf、视频、音频、zip、doc 等）

示例：本文位于 docs/网址工具等/组件使用.md，则图片放 docs/网址工具等/img/，表格放 docs/网址工具等/excel/，PDF 等附件放 docs/网址工具等/files/。

开发时资源直接从 docs/ 读取；构建时会复制到发布目录，无需放到 public/。

启动或构建前会自动校验资源（空文件、无效 xlsx 会在终端警告；CI 环境下会导致构建失败）。

Obsidian 双链语法总览 ​

写法解析为说明
![[file.png]]图片同级 img/
![[file.xlsx|Sheet]]表格同级 excel/，| 后为工作表名
![[file.xlsx|@svg:Sheet]]SVG 流程图从 xlsx 读取节点/连线并渲染
![[file.pdf]]PDF 嵌入同级 files/
![[clip.mp4|说明]]视频同级 files/
![[audio.mp3]]音频同级 files/
![[archive.zip|下载包]]附件链接同级 files/
[[其他文章]]站内链接同目录或 docs/ 下的 .md 页面
[[file.xlsx|Sheet]]表格链接式，效果同嵌入
[[file.zip]]附件下载链接式

不支持在 Markdown 中手写 <tableUtils urls='...'> 或 <imgUtils urls='...'>，请统一使用 Obsidian 语法。

矢量图 svgUtils ​

支持两种用法：手写 SVG 或 从 xlsx 数据文件自动生成流程图（节点、颜色、文本、顺序、连线）。

手写 SVG ​

在 Markdown 中直接使用 Vue 组件标签，将 SVG 代码放在组件内部。

示例代码
html
<svgUtils>
  <svg xmlns="http://www.w3.org/2000/svg" width="400" height="200" viewBox="0 0 400 200">
    <rect x="10" y="10" width="120" height="40" fill="#fff" stroke="#000"/>
    <text x="30" y="35">示例</text>
  </svg>
</svgUtils>

xlsx 数据格式 ​

节点表（工作表名可为 nodes、节点 或任意名称）：

列名（中英文均可）说明
类型 / typenode 或 节点（可省略）
id / 编号节点唯一 ID
文本 / label显示文字
填充 / fill在 Excel 中设置该列单元格背景色（不读取文本）
边框 / stroke在 Excel 中设置该列单元格边框/背景色
文本色 / textColor在 Excel 中设置该列或「文本」列的字体颜色
顺序 / order层级顺序（一列一层）；相同 order 的节点在同一列上下并列
x、y坐标（省略则按列自动排列）
宽、高 / width、height节点尺寸
形状 / shaperect、rounded（默认圆角矩形）
方向 / direction首行数据下可填：horizontal（默认，从左到右分列）、vertical（从上到下）

前置节点表（可选，工作表名 prerequisites、前置、前置节点）：

列名说明
id / 节点 / to当前节点 ID
predecessors / 前置节点 / from单个前置节点 ID（一行一条关系）

示例：review 的前置为 validate；end 需要 publish 与 archive 两条前置时，写两行即可。

无需添加分流、汇合、中转等辅助节点，只写实际业务节点，前置关系保持一行一条（一对一）。

默认从左到右按列展示：order 相同者在同一列上下并列，单列节点（如「开始」）垂直居中；连线为曲线；根节点位于画布垂直中央。

flow-demo.xlsx 示例：受理 下分 解析/扫描 两支，各自延续到 校验/补全 → 审核/通知/审计 → 发布/归档 → 结束。

连线表（可选，工作表名 edges、连线）：

列名说明
类型edge 或 连线
来源 / from起点节点 id
目标 / to终点节点 id
顺序 / order连线绘制顺序
文本 / label连线上的说明
边框 / stroke在 Excel 中设置线条颜色

连线优先级：连线表 > 前置节点表 > 按顺序自动生成（相邻顺序组之间全连接）。

颜色请在 Excel 里直接设置单元格格式（填充色、字体色、边框色），不要在单元格中输入 #RRGGBB 文本。

  • 工具栏：缩小 / 放大 / 重置；点击 放大 进入页内全屏（与表格相同），支持滚轮缩放、拖拽平移，按 Esc 关闭
