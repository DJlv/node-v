# 测试说明

本目录用于验证 VitePress 站点的**自动生成导航**功能。

## 测试项一览

| 场景 | 路径 | 预期行为 |
|------|------|----------|
| 同名单文件 | `single-file/` | 侧边栏直接链接，不重复嵌套 |
| 仅有 index | `index-only/` | 首页卡片 + 侧边栏均为单链接 |
| 多篇 md | `multi-file/` | 二级分组下展示多个文章链接 |
| 中文目录名 | `中文目录/` | 正常显示中文标题 |
| 三级目录 | `group-java/` | 二级 → 三级 → 文章（同 learn/java 结构） |
| 嵌套二级 | `nested/` | module-a、module-b 等子模块 |
| 根目录文章 | `overview.md` | 与文件夹并列出现在首页 |

## 如何验证

1. 重启开发服务：`npm run docs:dev`
2. 顶部导航应出现 **test**
3. 打开 `/docs/test/home`，首页应显示上述所有子项卡片
4. 进入任意文章页，左侧栏应展示完整二、三级目录树
